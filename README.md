# Simple Task Manager

A complete task manager application built with vanilla JavaScript, HTML, and CSS. This project demonstrates professional front-end development skills and prepares for international job opportunities.

## Features
- Add, edit, and delete tasks
- Manage task status (To Do / In Progress / Done)
- Modal-based task creation
- Keyboard accessibility (ESC to close modal)
- Fully responsive UI
- LocalStorage support to persist tasks

## Tech Stack
- HTML5
- CSS3 (Flexbox for layout)
- JavaScript (Vanilla)
- Git & GitHub for version control
